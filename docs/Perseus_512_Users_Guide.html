3.2 Hardware Setup

Step 1: Insert WM8904 Click Board

+

Install the WM8904 Click board into MikroBUS Slot 1 (critical for BCLK routing). Ensure secure connection of all pins.

Important: The WM8904 MUST be in Slot 1 - the I2S clock routing depends on this specific slot.

Step 2: Connect Audio Source

+

Connect your audio source (smartphone, PC) to the WM8904 LINE IN jack using a standard 3.5mm stereo cable.

Step 3: Connect Audio Output

+

Connect headphones or powered speakers to the WM8904 HP OUT jack. Ensure volume is set to a moderate level initially.

Step 4: Connect USB

+

Connect the USB micro cable to the Curiosity Nano board, then to your PC. The board will be powered via USB and enumerate as a COM port.

4. Operating the Demo

Quick Start Guide

  1. Power On: Connect USB cable - board powers up, LED meter sweeps
  2. Connect Audio: Audio source to LINE IN, headphones to HP OUT
  3. Play Music: Start audio playback - hear processed audio, see level meter
  4. Experiment: Press buttons and touch sensors to toggle effects
  5. Advanced: Launch Python GUI for full parameter control

4.1 Power-Up Behavior

ConditionBehaviorResult
Normal Power-Up USB connected, no buttons held Standard audio mode, all effects at defaults
Engine Synth Mode Hold BTN1 during power-up AVAS V8 engine synthesis enabled, pot controls RPM

4.2 Basic Operation

Audio Pass-Through

By default, audio flows from LINE IN through all processing stages to HP OUT. The LED meter shows the current output level.

Button Controls

Press BTN1 for Mute, BTN2 for Treble boost, BTN3 for Bass boost. All buttons toggle on/off.

Touch Controls

Touch T1 for Click-Clack, T2 for Surround/Widen, T3 for Bass Enhancer. Touch to toggle.

RPM Control

When Engine Synth is enabled, the potentiometer controls engine RPM from 0 to 8190 RPM.

8. 8-Band Graphic Equalizer

8.1 Overview

The 8-band graphic equalizer provides independent gain control over 8 frequency bands, allowing precise tonal shaping of the audio signal. Each band is implemented using a biquad IIR filter optimized for the dsPIC33AK FPU.

9. Acoustic Echo Cancellation (AEC)

9.1 Overview

Acoustic Echo Cancellation removes unwanted echo in hands-free audio systems. When audio played through a speaker is picked up by a nearby microphone, the far-end listener hears their own voice delayed - this is acoustic echo.

Without AEC:
+----------+     Speaker      +----------+
| Far-End  | ----------------> |   Room   | <---- Near-End Talker
|  Caller  |                   |          |
|          | <--- Echo! ------ |   Mic    | ----> Picks up speaker
+----------+                   +----------+

With AEC:
+----------+     Speaker      +----------+
| Far-End  | -------+-------> |   Room   |
|  Caller  |        |         |          |
|          |        v         |   Mic    | -----+
|          |       AEC        +----------+      |
|          |        |                           |
|          | <------+ (Echo Removed)            |
+----------+                                    |
                    ^---------------------------+

9.2 NLMS Algorithm

The AEC uses the Normalized Least Mean Squares (NLMS) adaptive filter algorithm:

Parameter Symbol Default Description
Filter OrderN512Number of taps (64ms @ 8kHz)
Step Sizemu0.05Adaptation speed
Regularizationdelta1e-8Prevents division by zero

9.3 Performance Metrics (ERLE)

ERLE Value Performance Description
< 5 dBPoorEcho clearly audible
5-15 dBModerateEcho still noticeable
15-25 dBGoodEcho mostly eliminated
> 25 dBExcellentEcho imperceptible

9.4 Echo Simulation Mode

Testing Made Easy: The firmware includes an Echo Simulation mode that digitally creates echo by adding a 60ms delayed copy of the output back to the input. This allows AEC testing with just Line In audio - no physical feedback path needed.

10. AVAS Engine Sound Synthesis

10.1 Overview

The Acoustic Vehicle Alerting System (AVAS) feature synthesizes realistic engine sounds for electric vehicles. This addresses the safety concern that quiet EVs pose to pedestrians who rely on engine noise to detect approaching vehicles.

Enabling AVAS: Hold BTN1 during power-up to activate Engine Synth mode. The potentiometer then controls engine RPM.

10.2 Engine Types

Inline-5ENG_I55-cylinder inline engine
Inline-6ENG_I66-cylinder inline engine
V8ENG_V8V8 engine (default)

10.3 RPM Control

ParameterValue

10.4 Blip Feature

The engine synth includes a "blip" feature that simulates throttle blips with attack/hold/decay envelope:

PhaseDescription
AttackRapid RPM rise when triggered
HoldBrief plateau at peak RPM
DecayNatural RPM fall-off back to current setting

10.5 Technical Details

Synthesis Method: Wavetable-based synthesis with harmonic content that varies with RPM
Sample Rate: 48 kHz stereo output
Mixing: Engine sound is mixed with LINE IN audio (or replaces it if no input)

11. Additional Audio Features

Tone Controls

Independent 3-band parametric equalizer with Treble (8kHz), Mid (1kHz), and Bass (100Hz) controls, each with +/-12dB range.

Bass Enhancement

Psychoacoustic bass boost that generates harmonics perceived as the missing fundamental - extended bass on small speakers. Toggled via TOUCH3.

Stereo Widening

Virtual surround using mid-side processing with frequency-dependent widening for an expanded soundstage. Toggled via TOUCH2.

Click-Clack Synthesizer

Turn signal indicator sound synthesis for automotive applications. Produces realistic relay-click sounds. Toggled via TOUCH1.

13.1 Connection Settings

13.2 Command Format

Command:  <kind><module><name><hex_data>\n
Response: =<module><name><status><hex_data>\n

kind:   '*' = set command, '?' = query command
module: 'e' = EQ, 'a' = AEC
status: 0x80 = OK, 0x00 = Error
