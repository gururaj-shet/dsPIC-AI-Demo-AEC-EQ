Audio Signal Flow

Audio enters through the WM8904 codec (LINE IN), is processed by the dsPIC33AK512MPS512 through multiple DSP stages including EQ, AEC, bass enhancement, stereo widening, and optional engine synthesis, then outputs back through the WM8904 to headphones/speakers.

Key Processing Stages

  • TDM Input: 24-bit audio received via DMA
  • Float Conversion: Converted to 32-bit float for FPU processing
  • Tone Control: 3-band parametric EQ
  • 8-Band EQ: Graphic equalizer with biquad filters
  • AEC: Acoustic echo cancellation (512 taps)
  • Bass/Widen: Enhancement effects
  • Engine Synth: V8 AVAS sound generation
  • Click-Clack: Turn signal synthesis
  • TDM Output: Back to 24-bit for DAC

Physical Controls

  • BTN1: Mute (hold at startup for Engine mode)
  • BTN2: Treble boost
  • BTN3: Bass boost
  • TOUCH1: Click-Clack toggle
  • TOUCH2: Stereo widen toggle
  • TOUCH3: Bass enhancer toggle
  • POT: Engine RPM (0-8190)

Control Interface

Parameters are controlled via UART (230400 baud) using the Python GUI application or direct serial commands.
